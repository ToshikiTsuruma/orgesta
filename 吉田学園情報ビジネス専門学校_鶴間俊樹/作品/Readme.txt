○======== [遊び方] ===================○
・中心にある木を５分間襲い来る敵から守る。
・木を守りきればクリア、木が壊さればゲームオーバー。
・敵を倒すと木に特殊な効果を持つリンゴがなる。
・プレイヤーの攻撃も木に当たってしまうので注意。
・一定時間内に敵を倒し続けてコンボを増やすととスコアの倍率が増える。
・敵を倒して得たエネルギーを使い苗木から身代わりを生み出せる。

○======== [操作方法] =================○
※ゲームパッド接続時はキーボード入力が無効になります。

・ゲーム終了：ESC

【キーボード】
・決定：SPACE,ENTER
・選択切り替え：上下左右（矢印）
・移動：WASD
・カメラの回転：IJKL
・攻撃：U
・強攻撃：O
・インタラクト：E
・ポーズ：P

【ゲームパッド】 ※Xinputのみ対応
・決定：A, Start		
・選択切り替え：十字キー
・移動：左スティック
・カメラの回転：右スティック
・攻撃：Rボタン
・強攻撃：ZRボタン
・インタラクト：Xボタン
・ポーズ：START
